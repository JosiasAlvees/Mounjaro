# 💊 Mounjaro Info — Landing Page

> **Página informativa e educacional sobre o medicamento Mounjaro (Tirzepatida)**  
> Desenvolvida com foco em design moderno, responsividade e responsabilidade médica.

</div>

---

## 📋 Sobre o Projeto

Esta landing page foi desenvolvida como um projeto de **conscientização e informação** sobre o medicamento **Mounjaro (Tirzepatida)**, abordando de forma clara e responsável:

- O que é o medicamento e como funciona
- Indicações clínicas legítimas
- Riscos e efeitos adversos documentados
- O problema do uso abusivo e automedicação
- Preocupações levantadas por especialistas médicos

O projeto tem caráter **exclusivamente informativo**, baseado em literatura médica e diretrizes clínicas, sem substituir consulta profissional.

---

## 🎨 Visual & Design

O design foi construído com uma identidade visual forte e coesa:

| Elemento | Detalhe |
|---|---|
| **Paleta** | Roxo escuro `#1a0a2e`, Roxo claro `#c4b5fd`, Branco `#ffffff` |
| **Tipografia** | Playfair Display (títulos) + DM Sans (corpo) |
| **Estilo** | Dark luxury — refinado, profissional e moderno |
| **Responsivo** | Mobile, tablet e desktop |

---

## ✨ Funcionalidades

- 🌟 **Hero animado** com partículas canvas, orbs flutuantes e efeito parallax visual
- 📌 **Header fixo** com efeito glassmorphism que acompanha toda a rolagem
- 🃏 **Cards com efeito tilt 3D** ao passar o mouse
- 🔽 **Accordion interativo** na seção de perguntas frequentes
- 👁️ **Scroll reveal** — elementos surgem suavemente conforme o usuário rola a página
- 💬 **Botão flutuante do WhatsApp** com label animado no hover
- 📬 **Formulário de contato** com feedback visual animado ao enviar
- ⚠️ **Indicadores de severidade** de risco com cores e brilho dinâmico

---

## 🗂️ Estrutura das Seções

```
├── Header          → Navegação fixa com glassmorphism
├── Hero            → Banner principal com animações e partículas
├── O que é         → Explicação sobre a tirzepatida
├── Indicações      → Quando o medicamento é clinicamente indicado
├── Riscos          → Efeitos adversos categorizados por severidade
├── Uso Abusivo     → Dados e alertas sobre automedicação
├── Preocupações    → Visão dos especialistas médicos
├── FAQ             → Perguntas frequentes com accordion
├── Contato         → Formulário + WhatsApp
└── Footer          → Disclaimer legal e informações
```

## 🚀 Como Usar

**1. Clone o repositório**
```bash
git clone https://github.com/seu-usuario/mounjaro-landing.git
```

**2. Acesse a pasta**
```bash
cd mounjaro-landing
```

**3. Abra no navegador**
```bash
# Basta abrir o arquivo diretamente:
open index.html

# Ou use uma extensão como Live Server no VS Code
```

> Não requer instalação de dependências — é 100% HTML, CSS e JavaScript puro.
